h1. univesp-le

Pacote LaTeX para listas de exercícios da Univesp.

h2. Como utilizar

Copie os arquivos @univesp-le.sty@ e @univesp.jpg@ para a pasta de trabalho.
Veja um exemplo abaixo:

<pre>
\documentclass[a4paper]{article}
\usepackage[utf-8]{inputenc}
\usepackage{univesp-le}

\disciplina{Nome da disciplina}
\autor{Nome do professor-autor}
\aulas{1--2}

\begin{document}

  \maketitle

  \begin{exercicio*}\label{ex:qquer}
    Este é um exercício do portfólio.
  \end{exercicio*}

  \begin{exercicio}
    Este é um exercício ordinário fazendo referência ao exercício \label{ex:qquer}.
  \end{exercicio}

  % Use o ambiente "respostas" para agrupar as respostas.
  \begin{respostas}
    \begin{exercicio*}
      Resposta do exercício do portfólio: não aparece para o aluno (opção "aluno" do pacote),
      mas aparece para o mediador ("mediador") e para o professor ("professor", opção padrão).
    \end{exercicio*}
    ...
  \end{respostas}
  
  % Use o ambiente "resolucoes" para agrupar as resoluções.
  % Elas aparecerão apenas para os moderadores (opção "moderador") e para o professor ("professor").
  \begin{resolucoes}
    \begin{exercicio}
      Resolução de um exercício.
    \end{exercicio}
    ...
  \end{resolucoes}

\end{document}
</pre>

h2. Motivação

Este pacote foi criado com o intuito de simplificar as interações entre a equipe de produção da Univesp e os professores-autores usuários de LaTeX.

Para o professor-autor, a produtividade que o LaTeX permite atingir é muito superior àquela que se pode obter com _softwares_ como o Microsoft Word, Open Office etc. Por outro lado, para a equipe de _design_ gráfico o LaTeX é uma "caixa de surpresas" que não se encaixa no _framework_ dela.

Deste modo, o intuito deste projeto é fornecer ao professor-autor um estilo LaTeX que dê conta das configurações de _layout_ definidas pela Univesp, sem que ele tenha de se ocupar com isso, conforme preconiza o LaTeX.
